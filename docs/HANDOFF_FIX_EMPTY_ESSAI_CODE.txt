Fix for empty 'Code essai' column in LaboPage > Interventions

- The DB does contain essai_code for imported interventions.

What this package fixes
- backend/current_fastapi/api/interventions.py now:

  - parses observations JSON when present
  - exposes:

    - code_essai
    - essai_label
- frontend/react/src/pages/LaboPage.jsx is included unchanged from v5 for consistency
- backend/current_fastapi/api/essais.py is included unchanged from v5 for consistency

Expected result
- LaboPage > Interventions > 'Code essai' should now show values like PLD / CFE / etc. instead of empty cells.
