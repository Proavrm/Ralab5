Règle métier rappelée
Le refresh lit les codes déjà construits par entreprise (gsa, ehtp, nge_routes, etc.). Il ne doit pas reconstruire bêtement un code si le code final existe déjà. Les cas spéciaux comme RASTOP ne doivent pas être confondus avec un vrai affaire chantier.
